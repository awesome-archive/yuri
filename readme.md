Macro actions based StarCraft AI with Keras

![nn.svg](/images/nn.svg)

## Usage

### Build Running Environment

```sh
$ pipenv install
```

### Run game

```sh
$ pipenv run python game.py
```

### Train model

```sh
$ pipenv run python train.py
```
